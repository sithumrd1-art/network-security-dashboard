# 🛡️ AI-Based Network Security Dashboard

A Python and Streamlit-based network security analytics dashboard that analyzes network traffic, detects suspicious activity, and predicts potential attacks using machine learning.

## 🚀 Features

- Upload network traffic CSV files
- Analyze network traffic records
- Filter records by protocol and status
- Display total traffic and detected attacks
- Visualize protocol distribution
- Monitor network traffic over time
- Identify suspicious source IP addresses
- Calculate network security risk level
- Predict normal or malicious traffic using AI
- Download filtered CSV and summary reports
- Professional cyber-security dashboard design

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit
- Scikit-learn
- Random Forest Classifier
- Joblib
